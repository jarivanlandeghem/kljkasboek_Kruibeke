# 💡 Idee

Ik wil een programma schrijven waar je **alle uitgaven en inkomsten** kunt bijhouden van een **vereniging**.  
Deze verdeel je in **categorieën**, zodat je per categorie kunt zien wat er is **binnengekomen** of **uitgegaan**.  

Bijvoorbeeld:  

- We hebben dit jaar **€500 uitgegeven aan kilometervergoedingen**  
- We hebben **€300 winst gemaakt op onze wafelenbak**

---

## ⚙️ Functionaliteiten

- **Loginpagina**  
- **Lijst met transacties**  
  - Toevoegen / verwijderen / bewerken  
- **Categorieën-tab**  
  - Kies een categorie en krijg een overzicht van de transacties daarin  
- **Gebruikersbeheer**  
  - Users toevoegen of verwijderen  
- **CSV-import**  
  - Importeer transacties uit een CSV-bestand  
  - Categoriseer ze één voor één en voeg ze toe aan het systeem
